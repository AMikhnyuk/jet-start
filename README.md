Webix Jet Demo App
===================

### How to run

#### Linux
- run ```dotnet restore```
- run ```npm install```
- run ```ASPNETCORE_ENVIRONMENT=Development dotnet run```
- open ```http://localhost:5000```


#### Windows

- open project in visual studio
- restore dependencies
- start the project
- open ```http://localhost:5000```

For more details, check https://www.gitbook.com/book/webix/webix-jet/details
